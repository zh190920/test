# 用户画像与课程推荐系统

基于汇川技术公司用户行为数据构建用户画像，实现精准课程推荐。

## 安装

```bash
pip install -r requirements.txt
export OPENAI_API_KEY=your-key
```

## 运行

```bash
python app.py
```

访问 http://localhost:5000 查看前端。

## 测试

```bash
python -m pytest tests/
```

## 项目结构

- `src/`: 核心模块
- `data/`: 示例数据
- `templates/`: 前端模板
- `tests/`: 单元测试

标签树构建方法
基于已有的标注文件构件初始化标签树，完成以目标：
树的每层节点代表一个标签，最后一层保存该用户标签。同时存在另一个字典保存每一个树某个上个节点下该子节点的所有向量。
1. 初始化时树随机选择标签树结构进行推理，如果标签树结构有更新则将更新后的标签扔到模型里去推理，让大模型模仿推理生成，这一步是为了让模型根据属性推理出和标签树一致的结构，比如标签树某个层级变了，加了一个层级，那么下次推理就可以借鉴这个新的层级来生成了。
2. 用向量模型+关键词依次将用户每层标签和树里该层节点的进行相似度计算，相同的标签挂载到已有标签树下。
3. 相似的用户标签以完全匹配路径和最大匹配路径来匹配。完全匹配为了防止第一步有漏网之鱼，使用更强的关键词匹配和向量匹配加权得分，看看能不能匹配到已有的标签，最大匹配路径就是能匹配到的最大匹配路径在该路径下生成新的节点。防止树的节点变得越来越宽，比如故障诊断和故障排查应该是一个节点。
4. 用户精确度标签必须要利用相似度和精确标签进行融合，保证一致性，相似的进行合并。
5. 用户标签与树里节点都不相似，则重新构建节点。
6. 标签可修改，比如某一层级的标签需要修改，则可以批量增删改查
